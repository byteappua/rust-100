## 示例项目 `os-concepts`
本项目演示了 Rust 中的一些基本系统编程概念。

### 功能演示
- **环境变量**: 读取 `PATH` 等环境变量。
- **进程管理**: 使用 `Command` 执行系统命令 (`ls`)。
- **多线程**: 创建线程并等待结果。
- **内存概念**: 查看不同类型的内存占用大小。

### 运行代码
```bash
cd Day81-90/81.OSConcepts
cargo run
```
