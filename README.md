# Rust - 100天从新手到大师

> 这是一个受 [Python - 100天从新手到大师](https://github.com/jackfrued/Python-100-Days) 启发的 Rust 语言学习计划。

Rust 是一门赋予每个人构建可靠且高效软件能力的系统编程语言。它注重内存安全、线程安全和高性能。本项目的目标是通过100天的学习，帮助新手掌握 Rust 语言，并能够使用 Rust 进行实际的项目开发。

## 目录

### 第一阶段：Rust 语言基础 (Day 01 - 15)
- **Day 01**: 初识 Rust - 环境搭建、Cargo 使用、Hello World
- **Day 02**: 变量与数据类型 - 变量绑定、可变性、基本类型
- **Day 03**: 函数与控制流 - 函数定义、if/else、循环
- **Day 04**: 所有权 (Ownership) - Rust 的核心概念
- **Day 05**: 引用与借用 (References & Borrowing)
- **Day 06**: 切片 (Slices) 类型
- **Day 07**: 结构体 (Structs) - 定义与实例化
- **Day 08**: 枚举 (Enums) 与模式匹配 (Match)
- **Day 09**: 模块系统 (Modules) - 包、Crate、模块
- **Day 10**: 常用集合 - Vector, String, HashMap
- **Day 11**: 错误处理 - Result 与 Panic
- **Day 12**: 泛型 (Generics)
- **Day 13**: Trait - 定义共享行为
- **Day 14**: 生命周期 (Lifetimes) 基础
- **Day 15**: 自动化测试

### 第二阶段：进阶概念 (Day 16 - 30)
- **Day 16**: 闭包 (Closures)
- **Day 17**: 迭代器 (Iterators)

- **Day 19**: 智能指针 (Smart Pointers) - Box
- **Day 20**: 智能指针 - Rc 与 RefCell
- **Day 21**: 内部可变性模式
- **Day 22**: 引用循环与内存泄漏
- **Day 23**: 并发编程 - 线程
- **Day 24**: 消息传递 (Channels)
- **Day 25**: 共享状态并发 (Mutex, Arc)
- **Day 26**: Sync 与 Send Trait
- **Day 27**: 面向对象编程特性
- **Day 28**: 模式匹配进阶
- **Day 29**: 高级 Trait
- **Day 30**: 高级类型

### 第三阶段：宏与不安全代码 (Day 31 - 40)
- **Day 31**: Unsafe Rust 介绍

- **Day 34**: 过程宏 (Procedural Macros)
- **Day 35**: FFI - 与 C 语言交互

### 第四阶段：生态系统与异步编程 (Day 41 - 60)
- **Day 41**: 异步编程基础 (Async/Await)
- **Day 42**: Tokio 运行时介绍
- **Day 43**: 构建异步 Web 服务器 (Hyper/Axum)

- **Day 47**: 错误处理最佳实践 (Anyhow / Thiserror)

### 第五阶段：Web 开发实战 (Day 61 - 80)

- **Day 62-65**: 使用 Actix-web 或 Axum 构建 RESTful API

- **Day 82**: 文件系统操作

- **Day 84**: 网络编程 (TCP/UDP)

- **Day 91-100**: 综合项目开发 (自选题目，如：分布式系统、高性能服务器、游戏引擎组件等)

## 如何使用

建议按照天数顺序学习。每一天的文件夹中包含了当天的学习笔记、代码示例和练习题。
