# Day 02 练习题 - 变量与数据类型

## 🎯 练习目标

通过这些练习,你将:
- 熟练掌握 `let` 和 `mut` 的使用
- 理解并应用变量隐藏 (Shadowing)
- 熟悉 Rust 的基本数据类型操作

## 📝 练习列表

### 练习 1: 修复编译器错误 (基础)
**难度**: ⭐

**题目**:
下面的代码无法编译通过。请找出原因并修复它，使程序能正确输出 `x = 6`。

```rust
fn main() {
    let x = 5;
    println!("The value of x is: {}", x);
    x = 6;
    println!("The value of x is: {}", x);
}
```

**要求**:
- 修改 `x` 的声明
- 确保两次 `println!` 都能执行

**参考答案**: [查看答案](./solutions/exercise1.rs)

---

### 练习 2: 数据类型大杂烩 (基础)
**难度**: ⭐⭐

**题目**:
声明不同类型的变量来完成以下任务，并打印它们：
1. 一个有符号整数 `age`，值为 30。
2. 一个浮点数 `price`，值为 19.99。
3. 一个布尔值 `is_active`，值为 true。
4. 一个字符 `initial`，值为 'R'。

**示例输出**:
```
Age: 30
Price: 19.99
Is Active: true
Initial: R
```

**要求**:
- 显式标注变量类型（尽管 Rust 可以推断，但为了练习请标注）
- 格式化打印结果

**参考答案**: [查看答案](./solutions/exercise2.rs)

---

### 练习 3: Shadowing 魔法 (进阶)
**难度**: ⭐⭐⭐

**题目**:
利用 Shadowing 机制，不使用 `mut` 关键字，完成以下变量转换：
1. start: 定义变量 `x` 为字符串 "5"。
2. step 1: 将 `x` 转换为整数 5 (解析字符串)。
3. step 2: 将 `x` 加 1 变为 6。
4. step 3: 将 `x` 乘 3 变为 18。

**提示**:
- 字符串转整数可以用 `x.parse().expect("Not a number")`

**参考答案**: [查看答案](./solutions/exercise3.rs)

---

### 练习 4: 元组与解构 (进阶)
**难度**: ⭐⭐⭐

**题目**:
创建一个包含你的 `(姓名, 年龄, 身高)` 的元组。
1. 打印整个元组。
2. 使用解构 (`destructuring`) 将元组拆分为三个变量。
3. 单独打印解构后的变量。

**示例输出**:
```
Tuple: ("Alice", 30, 1.75)
Name: Alice, Age: 30, Height: 1.75
```

**参考答案**: [查看答案](./solutions/exercise4.rs)
