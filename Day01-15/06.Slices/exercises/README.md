# Day 06 练习题 - 切片 (Slices)

## 🎯 练习目标

通过这些练习,你将:
- 理解切片 (`Slice`) 的概念：引用集合中一段连续的元素序列。
- 掌握字符串切片 (`&str`) 的用法。
- 掌握数组切片的用法。
- 理解切片如何解决所有权问题（不持有所有权）。

## 📝 练习列表

### 练习 1: 字符串切片基础 (基础)
**难度**: ⭐

**题目**:
给定字符串 `let s = String::from("Hello World");`
请创建两个切片：
1. `hello` 包含 "Hello"
2. `world` 包含 "World"

并打印它们。

**要求**:
- 使用 `[start..end]` 语法。
- 也可以尝试使用简写 `[..end]` 或 `[start..]`。

**参考答案**: [查看答案](./solutions/exercise1.rs)

---

### 练习 2: 数组切片 (基础)
**难度**: ⭐

**题目**:
给定数组 `let arr = [1, 2, 3, 4, 5];`
创建一个切片 `slice`，使其包含 `[2, 3]`。

**要求**:
- 注意索引是从 0 开始的。
- 验证切片内容是否正确。

**参考答案**: [查看答案](./solutions/exercise2.rs)

---

### 练习 3: 编写 `first_word` 函数 (进阶)
**难度**: ⭐⭐

**题目**:
编写一个函数 `first_word`，它接收一个字符串切片 `&str` 作为参数，并返回该字符串中的第一个单词（也是 `&str`）。
如果字符串中没有空格，则返回整个字符串。

**提示**:
- 需要将字符串转换为字节数组 `as_bytes()` 来遍历。
- 使用 `iter().enumerate()` 来获取索引和元素。
- 判断空格字符 `b' '`。

**测试用例**:
- 输入 `"Hello World"` -> 输出 `"Hello"`
- 输入 `"Rust"` -> 输出 `"Rust"`

**参考答案**: [查看答案](./solutions/exercise3.rs)

## 🏆 挑战任务

### 思考题: 中文切片
尝试运行以下代码，看看会发生什么？
```rust
fn main() {
    let s = "你好";
    let s1 = &s[0..1];
    println!("{}", s1);
}
```
**原因**: Rust 的字符串是 UTF-8 编码的。中文字符通常占用 3 个字节。尝试切片中间位置会导致 panic。
**修正**: 尝试修改索引为 `0..3` 看看能否正确打印 "你"。
