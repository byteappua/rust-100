# Day 03 练习题 - 函数与控制流

## 🎯 练习目标

通过这些练习,你将:
- 熟练编写带有参数和返回值的函数
- 掌握 `if` 表达式作为赋值语句的用法

## 📝 练习列表

### 练习 1: 数学函数 (基础)
**难度**: ⭐

**题目**:
编写一个函数 `is_even(n: i32) -> bool`，如果 `n` 是偶数返回 `true`，否则返回 `false`。
在 `main` 函数中调用它并打印结果。

**示例输出**:
```
Is 10 even? true
Is 7 even? false
```

**要求**:
- 必须使用函数
- 函数必须有返回值

**参考答案**: [查看答案](./solutions/exercise1.rs)

### 练习 2: 这个数大吗? (基础)
**难度**: ⭐⭐

**题目**:
使用 `if` 表达式（注意不是语句）来实现一个功能：
给定一个数字，如果大于 100，返回 "Big"，否则返回 "Small"。
将结果绑定到一个变量 `result` 并打印。

**示例代码结构**:
```rust
let n = 50;
let result = if ... { ... } else { ... };
println!("The number is {}", result);
```

### 练习 3: 1 到 N 求和 (进阶)
**难度**: ⭐⭐⭐

**题目**:
编写一个函数 `sum_1_to_n(n: i32) -> i32`，计算从 1 加到 n 的总和。
请尝试分别使用 `while` 循环和 `for` 循环实现两个版本。

**示例输出**:
```
Sum 1 to 5 = 15
Sum 1 to 100 = 5050
```

### 练习 4: 温度转换器 (进阶)
**难度**: ⭐⭐⭐

**题目**:
编写两个函数：
1. `celsius_to_fahrenheit(c: f64) -> f64`：将摄氏度转换为华氏度。公式：`F = C * 1.8 + 32`
2. `fahrenheit_to_celsius(f: f64) -> f64`：将华氏度转换为摄氏度。公式：`C = (F - 32) / 1.8`

**要求**:
- 使用 `f64` 类型
- 在 `main` 中测试几个值

## 🏆 挑战任务

### 综合挑战: 斐波那契数列生成器
**难度**: ⭐⭐⭐⭐

**题目**:
斐波那契数列是这样的：0, 1, 1, 2, 3, 5, 8, 13, 21...
第 0 项是 0，第 1 项是 1，之后的每一项都是前两项之和。

编写一个函数 `fib(n: u32) -> u32`，返回第 n 项斐波那契数。
提示：可以使用递归，也可以使用循环（循环效率更高）。

**挑战**:
尝试打印前 10 个斐波那契数。

**示例输出**:
```
Fib(0) = 0
Fib(1) = 1
Fib(2) = 1
...
Fib(10) = 55
```
